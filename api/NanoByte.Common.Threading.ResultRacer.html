IEnumerable<TInput> input

The input elements to try to produce outputs for.
Func<TInput, CancellationToken, TResult> factory

A function that takes an input element and a cancellation token (triggered when another call won the race) and returns a possible result. Return null to indicate that the function was unable to provide a result. Exceptions (except System.OperationCanceledException) are passed through to GetResult() and GetResultAsync().
