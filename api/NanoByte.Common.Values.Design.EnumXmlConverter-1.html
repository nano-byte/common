Examples

Add this attribute to the System.Enum:

[TypeConverter(typeof(XmlEnumConverter<NameOfEnum>))]

Remarks

System.Xml.Serialization.XmlEnumAttribute.Name is used as the case-insensitive string representation (falls back to element name).
