Parameters

factory Func<CancellationToken, T>

A function that takes a cancellation token (triggered when another call won the race) and returns a possible result. Return null to indicate that the function was unable to provide a result. Exceptions (except OperationCanceledException) are passed through to GetResult() and GetResultAsync().
