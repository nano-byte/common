Assumes two sorted lists. Determines which elements are present in newList but not in oldList.

public static IReadOnlyList<T> GetAddedElements<T>(this IReadOnlyList<T>? newList, IReadOnlyList<T>? oldList) where T : IComparable<T>, IEquatable<T>

Parameters

newList IReadOnlyList<T>

The new list of elements; can be null (will be treated as an empty list).

oldList IReadOnlyList<T>

The original list of elements; can be null (will be treated as an empty list).

Returns

IReadOnlyList<T>

A list of elements that were added.

Type Parameters

T

Remarks

Elements that are present in oldList but not in newList are ignored. Elements that are equal for System.IComparable<T>.CompareTo(T) but have been otherwise modified will be added.
